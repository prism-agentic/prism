# PRISM Landing Page — Design Brainstorm

## Context
PRISM is a self-evolving multi-agent framework for AI-assisted software development. The landing page needs to convey:
- Technical sophistication and innovation
- The "living system" concept — agents that learn and evolve
- The structured orchestration (pipeline phases, quality gates)
- Open-source credibility and developer trust

## Idea 1: "Neural Constellation" — Data Visualization Aesthetic

**Design Movement**: Generative data art meets developer tooling — inspired by Observable, Stripe's documentation, and astronomy visualization software.

**Core Principles**:
1. Information as decoration — every visual element encodes real data about the framework
2. Dark canvas with luminous accents — the "night sky" metaphor for a system that illuminates
3. Precision typography — monospace meets geometric sans for that "engineered" feel
4. Layered depth through subtle particle effects and connection lines

**Color Philosophy**: Deep space navy (#0a0e1a) as primary background, with cyan (#00d4ff) and warm amber (#ffb347) as dual accent colors representing "intelligence" (cyan) and "evolution" (amber). The duality reinforces the core concept of agents that learn.

**Layout Paradigm**: Full-bleed sections with asymmetric content placement. Hero uses a large animated node graph on the right (70% width) with text anchored left. Subsequent sections alternate between left-heavy and right-heavy layouts with diagonal dividers.

**Signature Elements**:
1. Animated connection graph showing agent relationships with pulsing data flow
2. Terminal-style code blocks with syntax highlighting that "type themselves"
3. Glowing orbs at section transitions representing evolution stages

**Interaction Philosophy**: Scroll-driven reveals with parallax depth. Elements emerge from darkness as the user scrolls, mimicking the "discovery" process of the framework itself.

**Animation**: Smooth entrance animations (fade-up with slight scale), connection lines that draw themselves, subtle floating particles in the background. All animations are performant CSS/framer-motion based.

**Typography System**: "JetBrains Mono" for code and technical labels, "Space Grotesk" for headings, system sans-serif for body. Heading hierarchy uses weight contrast (700 for h1, 500 for h2) rather than just size.

## Idea 2: "Blueprint Machine" — Industrial Technical Drawing Aesthetic

**Design Movement**: Technical blueprint / architectural drawing meets modern SaaS — inspired by engineering CAD software, Figma's marketing, and Swiss design principles.

**Core Principles**:
1. Grid as gospel — every element snaps to a visible underlying grid system
2. Annotation-driven storytelling — callouts, measurement lines, and labels explain the system
3. Monochromatic with strategic color — mostly grayscale with one electric accent
4. Mechanical precision — sharp corners, thin lines, exact spacing

**Color Philosophy**: Off-white parchment (#f8f6f0) background with charcoal (#1a1a2e) text and electric blue (#2563eb) as the single accent. The blueprint metaphor uses faint grid lines in light blue (#e0e8f0). This creates a "technical document" feel that developers instinctively trust.

**Layout Paradigm**: Strict 12-column grid with visible gridlines as a design element. Content blocks are "annotated" with measurement markers and callout lines. Sections use horizontal rules with labeled endpoints (like architectural drawings).

**Signature Elements**:
1. Visible grid overlay with measurement annotations at key breakpoints
2. Callout lines connecting UI elements to explanatory text (like technical diagrams)
3. "Stamp" elements — version badges, status indicators styled as engineering stamps

**Interaction Philosophy**: Hover reveals additional annotation layers. Clicking expands "detail views" like zooming into a blueprint. Minimal animation — precision over motion.

**Animation**: Subtle line-drawing animations for callout connections. Grid lines fade in on scroll. Stamp elements have a satisfying "press" animation on hover. Everything feels mechanical and deliberate.

**Typography System**: "IBM Plex Mono" for all technical content and navigation, "IBM Plex Sans" for body text. Headings use all-caps with generous letter-spacing (0.1em) for that engineering document feel.

## Idea 3: "Living Prism" — Organic Light Refraction Aesthetic

**Design Movement**: Organic futurism meets glassmorphism — inspired by Apple's visionOS, linear.app, and bioluminescent organisms. The "prism" in the name becomes the central visual metaphor.

**Core Principles**:
1. Light as the protagonist — gradients, refractions, and spectral effects tell the story of transformation
2. Organic geometry — rounded forms, flowing curves, and soft edges contrast with the technical content
3. Layered glass — frosted glass cards float above rich gradient backgrounds
4. Breathing rhythm — the page feels alive through subtle ambient animations

**Color Philosophy**: Rich dark background (#0c0c14) with spectral gradient accents that shift through the prism spectrum — from deep violet (#7c3aed) through electric blue (#3b82f6) to emerald (#10b981) and warm gold (#f59e0b). Each color maps to a framework concept: violet=orchestration, blue=intelligence, emerald=evolution, gold=quality.

**Layout Paradigm**: Flowing vertical narrative with overlapping glass cards. Hero section uses a massive 3D prism visual that refracts light into the page's color scheme. Content sections use staggered card layouts with varying depths (blur levels, shadow intensities).

**Signature Elements**:
1. Central prism hero element with animated light refraction creating the color palette
2. Frosted glass cards with spectral border gradients that shift on hover
3. Flowing gradient mesh backgrounds that subtly animate between sections

**Interaction Philosophy**: Mouse-responsive light effects — the prism follows cursor position slightly. Cards lift and glow on hover. Scroll triggers smooth gradient transitions between sections.

**Animation**: Ambient gradient mesh animation (slow, 20s+ cycles), card entrance with spring physics (framer-motion), hover glow effects with spectral color shifts. Background particles drift slowly like bioluminescent organisms.

**Typography System**: "Outfit" for headings (geometric but with organic roundness), "Inter" at 400/500 weights for body (readable, neutral). Display text uses gradient fills matching the spectral theme. Large hero text at 4-5rem with tight line-height.
